🗺️ Dufour-app – Versatile Web GIS

Dufour-app is a web-based GIS platform for mapping, built on QGIS Web Client 2 (QWC2). Upload your QGIS projects and visualize them directly in the browser.

Getting Started

  1. Use the Theme Switcher (menu → Themes) to select a map project.
  2. Navigate the map using mouse drag, scroll to zoom, or use the zoom buttons.
  3. Click on features to identify them, or use the region-identify tool.
  4. Use the Layer Tree (menu → Layers) to toggle layer visibility.

Uploading Projects

Use the REST API to upload QGIS project files (.qgs / .qgz): 

POST /api/v1/projects
Content-Type: multipart/form-data
Body: file=<your_project.qgz>

After upload, the project will appear in the Theme Switcher.
